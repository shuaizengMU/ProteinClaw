B — 输入 / 触发弹出菜单

用户输入 / 时，输入框上方弹出命令列表，支持上下键选择、继续输入过滤。
适合：命令多、不想占常驻空间、类似 VS Code 命令面板体验

C — ? 键打开帮助 Modal

按 ? 弹出全屏帮助覆盖层，列出所有命令和快捷键。
适合：完整文档、不占常驻空间、类似 vim 的 :help
